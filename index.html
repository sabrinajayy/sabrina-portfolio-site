Scrambld

Scrambld is a simple game built with Ruby on Rails. Users are given 10 random letters and are to type the longest word he/she can find. Score is calculated based on time completed and word length

It was built using Ruby on Rails, HTML, CSS, and Bootstrap for mobile responsiveness.

Each word submitted is verified as an actual English word by using a dictionary API
